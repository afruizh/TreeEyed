Simple Inference
===================================

The following example shows the inference process on an **RGB raster image** to identify trees in a region of interest.
**HighResCanopyHeight** model was employed and different type results were generated (tree binary mask, tree grayscale mask, tree shapes, tree bounding boxes and tree centroids)
